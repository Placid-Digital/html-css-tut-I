This system is created to manage HTML tags and JavaScript used for tracking the websites.

Google tag manager initial release was on 1st October 2012. Google tag manager version 2 was released on October 2014.

The Tag is nothing but the website code that help us to measure the website traffic and information about the visitor behavior, but adding too many tags makes your website slow.

Google tag manager helps the site owners to update the website quickly and easily.

With the help of Google tag manager user interface, you can update Ad Words and analytics instead of editing the code in the website.

Why Tag Management system?
If your site is very slow and burdensome due to many tags, taking more time to implement new tags, Google Tag manager helps you in making your job easier.

Managing multiple tags is a tedious process but with the help of Google Tag Manager you can consolidate all your site tags in a single Admin panel. Now you can include that snippet code in your website.

One of the subsidiary feature in Google Tag manager is, Asynchronous Tag Loading.

Asynchronous tag loading is helpful in preventing the slow-loading tag from slowing down the rest of your website.

How to set up Google Tag Manager?
1. First visit the Google Tag manager https://www.google.com/analytics/tag-manager/
2. Set up your account name
3. In second step set up your container name
4. You have to choose the container type that is the web, iOS or android
5. Accept the terms of the agreement
6. Finally, you will receive Google Tag manager code
7. Install the code on your website.

These are the steps to set up Google tag manager for the Web.

Next let us see about how to set up Google Tag manager for Mobile Apps.

We have to follow the same steps like Google Tag manager for web but instead of choosing web we have to implement Google Tag manager SDK for Android or iOS.

After the setup process, you can add tags such as Google ad words, Analytics and conversion tracking tags through Google Tag manager user interface.

By this way you can implement Google Tag Manager for your website.

In this blog, let us see about the five campaign settings that the new advertisers need to know about.

1. Stick with “SEARCH NETWORK ONLY”.
2. Choose “ALL FEATURES”.
3. Expand “LOCATION OPTIONS”.
4. Expand “AD DELIVERY:AD ROTATION”.
5. Understand Google’s “SEARCH PARTNERS” option.

Now let us see about the five campaign settings in detail.

1. STICK WITH “SEARCH NETWORK ONLY”:

When you are setting up your network campaign in Google Adwords it is best to avoid using the option “search network with display select” but Google says it is the best option to choose.

In Google “search network with display select” option if you are not using the right option to control your budgets it will spend your budget rapidly.

So it is better to choose “SEARCH NETWORK ONLY” option while setting up the campaign.

2. CHOOSE “ALL FEATURES”:

After choosing the search network campaign, you have to choose the type, i.e. standard or all features. If you are choosing standard you can use only limited options.

Standard option includes location, site links, call. You can use only this option but in the case of ALL FEATURES you can opt for call outs, reviews, app, structured snippets etc..

In all features you can also choose your ad scheduling, ad start date and end date etc.

3. EXPAND “LOCATION OPTIONS”:

Google does not display the location options in expanded form. Location settings can be expanded by clicking on the advanced option.

In location settings we can see the target and exclude option.

On target and exclude section there is some option to choose which type of audience we need for our business.
We can use our audience based on their area, their interest in our products etc..

4. EXPAND “AD DELIVERY:AD ROTATION”:
Google does not display expanded delivery option. In expanded form, we can see the following options.
1. Optimize for clicks
2. Optimize for conversions.
3. Rotate evenly
4. Rotate indefinitely

The default setting the Adwords account set is “optimize for clicks”. This setting helps in generating more clicks for your ads.

Before choosing the ad delivery and ad rotation setting be aware of your campaign settings and goals.

5. UNDERSTAND GOOGLE’S “SEARCH PARTNERS” OPTION:

In Google’s default setting search option is very important. By default, search partners are included in the search network campaigns, but if you want your ads to not show on search partners deselect the “include search partners” option.

These are the campaign settings new advertisers should know about Google Adwords.
